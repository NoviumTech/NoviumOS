# NoviumOS Style Guide

## Philosophy

NoviumOS should feel calm, modern, and thoughtfully designed.

Every animation, color, and interface element should have a purpose.

## Personality

- Minimal
- Friendly
- Fast
- Elegant
- Consistent

## Typography

Primary
- Inter

Monospace
- JetBrains Mono

## Color Palette

Primary Accent
- Blue

Background
- Dark navy

Text
- White

Secondary Text
- Soft gray

## Corners

Standard Radius
- 16px

## Animations

- Smooth
- Purposeful
- Never distracting

## Icons

- Simple
- Rounded
- Consistent stroke width

## Design Principles

- No unnecessary clutter
- Every feature should solve a problem
- Beauty should never reduce usability
