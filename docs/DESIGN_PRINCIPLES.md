# NoviumOS Design Principles

## 1. Simplicity First

If a feature can be removed without harming usability, remove it.

## 2. Every Animation Has Meaning

Animations should communicate movement or state changes.

## 3. Consistency

The same action should always produce the same result.

## 4. Calm Interface

Avoid unnecessary visual noise.

## 5. Fast By Default

Users should never wait because of unnecessary effects.

## 6. Accessibility

Everything should remain usable with keyboard navigation and screen readers.

## 7. Respect The User

No ads, forced accounts, unnecessary notifications and no telemetry enabled by default. :D
